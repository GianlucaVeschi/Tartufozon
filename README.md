# Tartufozon
Mobile Android App to connect truffles retailers in Italy with customers in Germany and the rest of the EU.

## Tech Stack
- Kotlin
- MVVM
- Jetpack Compose
- Dagger Hilt
- Retrofit / OkHttp
- Room DB
- Junit5
